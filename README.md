
<h1>Cyber Hygiene Scorecard</h1>

<h2>Description</h2>
Cyber Hygiene Scorecard – Control Maturity & AI-Driven Risk Profiling Dashboard
Developed a Streamlit-based cybersecurity assessment platform aligned to the Rohan Cybersecurity Maturity Model and NCSC Cyber Assessment Framework (CAF). The solution standardizes questionnaire data, generates normalized control maturity scores, and applies a hybrid machine learning approach combining Random Forest classification (cross-validated using Stratified K-Fold) and Isolation Forest anomaly detection to identify abnormal risk patterns. The system provides AI-driven predictions, confidence scoring, feature importance insights, governance-stage recommendations, and performance evaluation metrics (precision, recall, F1-score, confusion matrix) to support audit readiness and risk-informed decision-making.
